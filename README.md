#  AirBnb's in Seattle

## Project Description

Our group will be working with the **AirBnb data collected on Seattle** as provided by `InsideAirBnb`. InsideAirBnb is “an independent, non-commercial set of tools and data that allows you to explore how Airbnb is really being used in cities around the world” (http://insideairbnb.com/about.html) The data used can be found at http://insideairbnb.com/get-the-data.html.
